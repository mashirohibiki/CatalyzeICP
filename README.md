# CatalyzeICP
一个可同过公司名、主备案号查子备案号的python小公举

## Title
一个可通过公司名、主备案号查备案号和子备案号的python工具（目前所调用的网站接口只支持单个ip每天查20次）

## Usage
#对应参数
```
-info	通过主备案号查询或通过公司名查询
-file	检索文件批量查询
```
